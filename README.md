
# CWUS

A Qt widget to control cmus.

# Features

- Basic Controls
- Slider Controls
- Live theme reload ($HOME/.config/cwus/colors.json)

## Themes Config Example

```json
import Component from 'my-project'

function App() {
  return <Component />
}
```
