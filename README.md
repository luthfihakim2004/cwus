
# CWUS

A Qt widget to control cmus.

# Features

- Basic Controls
- Slider Controls
- Live theme reload ($HOME/.config/cwus/colors.json)

## Themes Config Example

```json
{
  "accent": "#80316ec9",
  "darkaccent": "#3d88fa",
  "background": "#80030101",
  "foreground": "#ffff7b"
}
```
